# TextileAI 🧵

TextileAI is a high-fidelity, professional-grade textile engineering and design platform. Built for the modern weaver, it combines advanced simulation engines with a premium industrial design aesthetic to streamline your workflow from concept to machine production.

## 🚀 Key Features

- **Advanced Simulation Engine**: Real-time material physics and weave pattern simulation.
- **Dynamic Nozzle Management**: Multi-yarn, multi-nozzle sequence management for complex weaving.
- **Engineering-First UI**: Precision inputs for Warp and Weft engineering with instant visual feedback.
- **Production-Ready Exports**: Generate machine-readable files and professional PDF reports.
- **Materials Database**: Comprehensive library of fiber data and material physics.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS with modern aesthetics (glassmorphism, subtle animations)
- **Database/Auth**: Supabase
- **State Management**: Zustand
- **Deployment**: Vercel

## ⚙️ Getting Started

### Prerequisites

- Node.js (Latest LTS)
- npm or pnpm

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/iamyashkachhadiya-oss/TextileAI.git
    cd TextileAI
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Set up environment variables**:
    Create a `.env.local` file and add your Supabase credentials:
    ```env
    NEXT_PUBLIC_SUPABASE_URL=your_project_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
    ```

4.  **Run the development server**:
    ```bash
    npm run dev
    ```

5.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
